## Overview

Thyroid Check is a Streamlit-based web app that takes in patient information and uses a trained machine learning model to predict whether a patient is likely to experience a recurrence of thyroid disease. 
This tool is intended to assist medical professionals in making informed decisions.

---

## Features

- Clean and interactive web interface built with Streamlit
- Accepts 16 clinical input features
- Instant recurrence prediction (Yes / No)
- Input validation to ensure all fields are filled before prediction
- Dropdown menus with human-readable labels
